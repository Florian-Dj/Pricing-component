# Pricing Component

## Installation 
⚠️ Node.js is require 

For install packages:
```
npm install 
```

For lunch the server:
```
npm run start
```
Server should be running at http://localhost


## Hosting
Run a build
```
npm run build
```
In the host select the dist folder
